2. **IAFD 403 errors** — STATE.md noted IAFD detail pages return 403. Still unresolved; no fix visible from this audit.

3. **update_stash_profile = false** — Face detection runs but never auto-sets profile photos. Intentional (pending validation) per config comment.

4. **missingScenes plugin disabled** — Is this intentional?

5. **SSH access broken** — The SSH keys (`unraid_codex_ed25519`, `unraid_ed25519`) have been removed from Unraid's authorized_keys. Re-adding a key via Unraid web UI → Settings → SSH would restore automated access.

6. **1077 items in Whisparr Activity queue** — Could indicate stuck downloads or a large pending import queue worth reviewing.
